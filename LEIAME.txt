A base de dados utilizada pelo sistema é o mongoDB em suas configurações padrões (localhost: 27017, sem autenticação).
Caso haja necessidade de mudança, é necessário escrever os dados do MongoDB no arquivo "database.json", localizado na pasta "/config".

O banco de dados e a collection utilizada chamam-se "person", e as chaves utilizadas são "name" e "cpf".

O serviço MongoDB é iniciado no Ubuntu através do comando: sudo systemctl start mongodb
Você também pode verificar se ele foi iniciado corretamente através de: sudo systemctl status mongodb

Antes de testar o programa, é necessário executar o script installer.py.
Ele é responsável por instalar os módulos dependentes e popular o banco de dados (LEMBRE-SE DE CONFIGURAR O database.json ANTES DISSO!).

Para executar o instalador: python installer.py

Para inicar o programa: python __init__.py

O index do sistema estará em http://localhost:5000/

ATENÇÃO: O FORMULÁRIO DE CONSULTA POSSUI MÁSCARAS! TODOS OS CPFs INSERIDOS DEVEM CONTER PONTOS E TRAÇOS.
